# Cloud スクリプトテンプレート指示

> **読み込み条件**: type === "cloud" の場合
> **対応ランタイム**: Bash, Python

---

## 目的

クラウドCLI操作を行うスクリプトを生成する。

---

## AIへの実装指示

### 必須機能

1. **リソース管理**
   - リソース作成/削除
   - リソース一覧取得
   - タグ管理

2. **ストレージ操作**
   - オブジェクトアップロード/ダウンロード
   - バケット操作
   - 同期処理

3. **サーバーレス**
   - 関数デプロイ
   - 関数呼び出し
   - ログ取得

4. **認証管理**
   - 認証情報の読み込み
   - プロファイル切り替え

### 推奨依存関係

| ランタイム | 推奨パッケージ |
|------------|----------------|
| Bash | aws-cli, gcloud, az |
| Python | boto3, google-cloud-*, azure-* |

### 環境変数

- `{{CLOUD_PROVIDER}}`: クラウドプロバイダー（aws/gcp/azure）
- `{{CLOUD_REGION}}`: リージョン
- `{{CLOUD_PROFILE}}`: 認証プロファイル

### 引数仕様

| 引数 | 必須 | 説明 |
|------|------|------|
| --provider | ○ | クラウドプロバイダー |
| --action | ○ | 実行するアクション |
| --resource | △ | リソース識別子 |
| --region | × | リージョン |
| --profile | × | 認証プロファイル |

### 出力形式

- 操作結果をJSON形式で出力

---

## コード構造の指示

AIは以下の構造でコードを生成してください:

```
1. インポート/設定
   - 認証情報読み込み
   - リージョン設定

2. ユーティリティ関数
   - authenticate(): 認証処理
   - executeCommand(): CLIコマンド実行
   - parseResponse(): レスポンス解析
   - handleError(): エラー処理

3. プロバイダー別関数
   - aws_*(): AWS操作
   - gcp_*(): GCP操作
   - azure_*(): Azure操作

4. メイン処理
   - 引数パース
   - 認証
   - アクション実行
   - 結果出力
```

AIは以下の基準を満たすコードを生成してください:

- [ ] 認証情報の安全な管理
- [ ] レート制限の考慮
- [ ] コスト意識のある操作
- [ ] リソースの適切なタグ付け
- [ ] エラーの詳細な報告
