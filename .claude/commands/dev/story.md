# /dev:story - ストーリー → タスク生成コマンド

## 概要

ユーザーストーリーからTDD/E2E/TASK分岐付きタスクリスト（task-list.json）を生成します。

## 使い方

### 引数付き

```
/dev:story ログインフォームを作成する。メールアドレスとパスワードでログインできるようにする
```

### 引数なし

```
/dev:story
```

→ 対話的にストーリーを取得、または `docs/USER_STORIES.md` から読み込み

## 実行方法

**⚠️ 重要: このコマンドは必ず `dev:story` スキルを発火して実行してください。**

### 必須手順

1. **スキルを発火**: スキルファイルの指示に従って実行する
2. **直接実装を禁止**: スキルを発火させずに、このコマンドファイルの内容から直接実装を開始してはならない
