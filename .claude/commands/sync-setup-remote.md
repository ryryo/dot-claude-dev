# /sync-setup-remote - セットアップスクリプト同期コマンド

## 概要

`dot-claude-dev` の `setup-claude-remote.sh` テンプレートの共有セクションを、
外部プロジェクトにコピーされたスクリプトに同期する。
プロジェクト固有のカスタマイズ（マーカー外の部分）は保持される。

## 使い方

```
/sync-setup-remote
```

## 実行方法

**必ず `sync-setup-remote` スキルを発火して実行してください。**

1. **スキルを発火**: スキルファイルの指示に従って実行する
2. **直接実装を禁止**: スキルを発火させずに、このコマンドファイルの内容から直接実装を開始してはならない
