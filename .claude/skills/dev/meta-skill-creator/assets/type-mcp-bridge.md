# MCP Bridge スクリプトテンプレート指示

> **読み込み条件**: type === "mcp-bridge" の場合
> **対応ランタイム**: Node.js (TypeScript)

---

## 目的

MCPサーバーとの連携を行うスクリプトを生成する。

---

## AIへの実装指示

### 必須機能

1. **MCPサーバー接続**
   - stdioトランスポート
   - SSEトランスポート
   - 接続状態管理

2. **ツール呼び出し**
   - ツール一覧取得
   - ツール実行
   - 結果処理

3. **リソースアクセス**
   - リソース一覧取得
   - リソース読み込み

### 推奨パッケージ

- `@modelcontextprotocol/sdk`

### 環境変数

- MCPサーバー固有の設定

### 引数仕様

| 引数 | 必須 | 説明 |
|------|------|------|
| --server | ○ | MCPサーバー名/パス |
| --tool | △ | 実行するツール名 |
| --args | △ | ツール引数（JSON） |
| --resource | △ | 取得するリソース |

---

- [ ] サーバー起動失敗時の適切なエラー
- [ ] ツール実行タイムアウト
- [ ] 結果の構造化出力
- [ ] 接続の確実なクローズ
