## レポートフォーマット: api-integration（API/SSE連携確認）

### 各ステップで必ず実行するコマンド
- API呼び出しトリガー後に `console` を実行し、**出力を全文引用**
- `errors` を実行し、**出力を全文引用**
- 状態変化ごとに `snapshot` を実行し、**出力を引用**
- スクリーンショットは補助（最小限でよい）

### レポート構造

```
## 1. 初期状態
### snapshot
[出力引用]

## 2. API呼び出し後
### snapshot（状態変化）
[出力引用]

### console 全文
[出力を全文引用]

### errors 全文
[出力を全文引用、なければ「エラーなし」]

## 3. 最終状態
### snapshot
[出力引用]

## 4. スクリーンショット（補助）
- {パス}: {説明}
```
