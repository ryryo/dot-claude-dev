## レポートフォーマット: responsive（レスポンシブ確認）

### 各ステップで必ず実行するコマンド
- 各 viewport で `snapshot -i` を実行し、**出力を引用**
- 各 viewport で `screenshot {SCREENSHOT_DIR}/{NN}-{device}.png` を保存
- viewport サイズ: モバイル(375x667)、タブレット(768x1024)、デスクトップ(1280x800)

### レポート構造

```
## モバイル (375x667)
### snapshot -i
[出力引用]
### スクリーンショット
- {パス}

## タブレット (768x1024)
### snapshot -i
[出力引用]
### スクリーンショット
- {パス}

## デスクトップ (1280x800)
### snapshot -i
[出力引用]
### スクリーンショット
- {パス}

## viewport 間の差異
- [要素配置の違いや問題点]
```
