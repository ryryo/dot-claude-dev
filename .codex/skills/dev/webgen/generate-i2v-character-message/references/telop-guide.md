## 基本方針

Webサイトでは、テロップは動画ファイルへ焼き込まずHTML/CSSで重ねる方が扱いやすい。文言変更、レスポンシブ、アクセシビリティ、ABテストが楽になる。

動画単体で確認、共有、配布する場合だけ`burn_telop.py`で焼き込む。

## 既定スタイル

黄色帯を使う既定値:

- 位置: 下部2段
- 背景: 黄色 `#FFF176`、不透明度0.97
- 文字: 太い黒 `#17120F`
- 縁取り: 白、6px
- フォント: macOSではHiragino Sans W9を優先
- 1行目: `サブスクで明朗会計`
- 2行目: `勧誘一切無し`

## 調整項目

```bash
python3 <skill>/scripts/burn_telop.py input.mp4 \
  --line1 "サブスクで明朗会計" \
  --line2 "勧誘一切無し" \
  --bg-color FFF176 \
  --text-color 17120F \
  --border-color FFFFFF \
  --font-scale1 0.083 \
  --font-scale2 0.089 \
  --y1-ratio 0.594 \
  --y2-ratio 0.761
```

比率指定は動画の高さに対する割合。人物の顔、手、商品を隠す場合は`--y1-ratio`と`--y2-ratio`を下げすぎない。

## HTMLテロップとの使い分け

- Web本番: HTML/CSSで重ねる。
- 比較用: 焼き込み動画を作る。
- SNSや動画単体配布: 焼き込み動画を作る。

焼き込み後も動画だけに重要情報を依存させない。近接するHTML本文または代替テキストにも同じ意味を置く。
