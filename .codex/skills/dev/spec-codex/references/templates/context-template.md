# context.md テンプレート

`docs/PLAN/{YYMMDD}_{slug}/references/context.md` は、この PLAN 固有の確認済み事実を記録する。
プロジェクト全体のルールや構成（CLAUDE.md / README に記載済みのもの）は対象外。

```markdown
# コンテキスト

## 関連ファイル

この実装で触る特定のファイル・モジュール・API を実際に確認した結果。

| パス | 確認した事実 | PLAN への影響 |
| --- | --- | --- |

## 外部参照

コードベース外の資料で `references/` にコピーまたは要約したもの。

| 情報源 | ローカル参照パス | 確認した事実 |
| --- | --- | --- |

## コマンド

AC や review ゲートで使う検証コマンド。実在確認済みのものだけ記録する。

| コマンド | 用途 | 使用 Step / AC |
| --- | --- | --- |

## 除外した前提

PLAN の前提として採用しなかったファイル・API・コマンド・挙動。

| 前提 | 除外した理由 | 採用に必要な根拠 |
| --- | --- | --- |
```

## 契約

- `関連ファイル` はこの PLAN で変更・参照するファイルを実際に Read / Grep して確認した事実だけを書く。
- `外部参照` はコードベース外の資料のみ。コードベース内ファイルは Gate の `constraints` / `affectedFiles` で参照する。
- `コマンド` は AC や review で使うコマンドの実在を事前確認したもの。
- `除外した前提` は未確認の API・ファイル・コマンドを PLAN の前提から外す根拠を残す。
