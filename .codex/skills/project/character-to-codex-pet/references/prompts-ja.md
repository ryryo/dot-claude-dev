# 生成プロンプト

## 基準画像

以下をキャラクター固有情報で補完し、既存画像を編集対象として使う。

```text
用途: style-transfer
成果物: Codexアニメーションペット用の正準キャラクター画像

主要求:
添付した既存キャラクターを、制作素材として使える本格的なピクセルアートへ変換する。

必ず維持するもの:
- 同じキャラクターとして識別できる顔と表情
- 身体の比率とシルエット
- 髪、耳、角、尻尾などの識別特徴
- 服装、配色、模様、重要な小物

スタイル:
- 手作業で打ったような明確なピクセルクラスタ
- 硬く鮮明なピクセル境界
- 制限された一貫性のあるカラーパレット
- 小さなUIサイズでも読める輪郭
- ピクセル密度、輪郭線、陰影方法を全身で統一する
- アンチエイリアス、滑らかなグラデーション、絵画的質感を使わない
- 高解像度イラストへ単純なピクセルフィルターを重ねた見た目にしない

構図:
- キャラクター1体だけ
- 全身を中央に配置する
- ニュートラルなidleポーズ
- 身体の周囲に十分な余白を取る
- 身体や小物を切らない

背景:
- 指定された単一のクロマキー色だけを使う
- 床、風景、グラデーション、模様、影、反射を入れない

禁止:
- キャラクターの再設計
- 追加キャラクター
- 文字、ロゴ、透かし
- 吹き出し、記号、スピード線
- 接地影、投影、光彩
```

## 修正

```text
前の候補を編集対象として使う。
承認済みの方向性、キャラクター同一性、ピクセル密度、配色、輪郭、服装、体型、ポーズは維持する。

今回変更するもの:
- <変更を1つだけ記載>

変更しないもの:
- <固定条件を列挙>

それ以外を再解釈、追加、削除しない。
```

## 差分生成

run内の状態別プロンプトへ、必要に応じて次を追加する。

```text
添付したcanonical-base画像を、キャラクター外観の唯一の正解として扱う。

全フレームで厳密に維持するもの:
- キャラクターの同一性
- 顔、身体比率、シルエット
- ピクセルアート技法とピクセル密度
- カラーパレット
- 輪郭線の色と太さ
- 服装、模様、小物
- 陰影方法

変更するのは指定されたアニメーション状態のポーズと表情だけにする。
キャラクターを再設計しない。
レイアウトガイドは配置だけの参照とし、枠線、ラベル、ガイド色を出力へ描画しない。
```
