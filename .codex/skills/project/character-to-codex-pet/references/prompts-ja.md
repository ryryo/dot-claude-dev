# 生成プロンプト契約

## 基準画像

```text
用途: style-transfer
成果物: Codexアニメーションペット用の正準キャラクター画像

添付した既存キャラクターを、制作素材として使える本格的なピクセルアートへ変換する。

必ず維持するもの:
- 同じキャラクターとして識別できる顔、表情、身体比率、シルエット
- 髪、耳、角、尻尾などの識別特徴
- 服装、配色、模様、重要な小物

スタイル:
- 手作業で打ったような明確なピクセルクラスタ
- 硬く鮮明なピクセル境界、限定色、一貫したピクセル密度
- 小さなUIでも読める輪郭と単純なクラスタ陰影
- アンチエイリアス、滑らかなグラデーション、絵画的質感を使わない
- 高解像度イラストへピクセルフィルターを重ねただけの見た目にしない

構図:
- 1体だけ、全身、中央、ニュートラルなidleポーズ
- 身体と小物を切らず、周囲に安全余白を取る

背景:
- 指定された単一クロマキー色だけ
- 床、風景、グラデーション、模様、影、反射を入れない

禁止:
- 再設計、追加キャラクター、文字、ロゴ、透かし
- 吹き出し、記号、スピード線、接地影、投影、光彩
```

## 基準画像の修正

```text
前候補を編集対象として使う。
承認済みの方向性、キャラクター同一性、ピクセル密度、配色、輪郭、服装、体型、ポーズを維持する。

今回変更するもの:
- <変更を1つだけ記載>

変更しないもの:
- <固定条件を列挙>

それ以外を再解釈、追加、削除しない。
```

## 全差分共通

run内の状態別プロンプトへ次を追加する。

```text
canonical-baseを外観の正解、canonical-cellをセル内サイズ・足元・ピクセル密度の正解として扱う。

全フレームで厳密に維持するもの:
- キャラクターの同一性、顔、身体比率、シルエット
- ピクセル技法、ピクセル密度、パレット、輪郭線、陰影方法
- 服装、模様、小物
- canonical-cellと同じ見かけサイズと足元基準

変更するのは指定状態のポーズと表情だけにする。
動きを見せるためにカメラを引かない、ズームしない、キャラクター自体を縮小しない。
ポーズの上下移動はセル内の小さな範囲に収める。
レイアウトガイドの枠、ラベル、色を出力へ描画しない。
```

## jumping追記

```text
全5フレームで頭、顔、胴体のピクセルスケールをcanonical-cellと一致させる。
ジャンプを見せるためにキャラクターを縮小したり、カメラを引いたりしない。
最終192x208セル換算の上下移動は約10px、許容範囲8〜16pxに収める。
予備動作と着地の屈みはポーズとして短くしてよいが、頭や身体そのものを拡大縮小しない。
影、床、埃、着地マーク、スピード線を使わない。
```

## failed追記

```text
うつむきや屈みは関節と姿勢だけで表現する。
屈んだフレームをセルいっぱいに拡大しない。頭部だけを大きくしない。
失敗中もcanonical-cellと同じピクセル密度を維持する。
浮遊する記号、赤いX、分離した涙や煙を使わない。
```

## 左右移動追記

```text
全フレームでcanonical-cellと同じ身長・ピクセル密度を維持する。
歩容による足元変動は最終セル換算12px以内にする。
移動を見せるためにズーム、縮小、スピード線、埃、影を使わない。
```
